# Nivelamento 👶🏾
Tivemos uma introdução bem clara e objetiva sobre Algoritmo, usando o software Portugol Studio. Desenvolvendo os conceitos relacionados à algoritmo, como: Lógica booleana (E, OU, NOT), Árvore de decisão, Estruturas de Laço, Conceito de recursividade, Estrutura de dados (Vetor e Matriz) e Conceito de Variável e Constante. E finalizando o módulo com alguns conceitos sobre Git e Github.

## Organização 📋
Os exercícios estão separados por listas, cada lista desenvolve um conceito. 

1. Entrada e Saída de Dados
2. Desvios Condicionais, Operadores Lógicos
3. Laços de Repetição (Enquanto, para ... até)
4. Funções e Recursividade 
5. Vetores e Matrizes
